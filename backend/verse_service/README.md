# Verse Service

Microserviço responsável por buscar e cachear versículos bíblicos.
Estrutura Clean Architecture:

```
/domain
  entities
  repositories
  usecases
/data
  datasources
  repositories_impl
/presentation
  controllers
/infrastructure
  framework_adapters
```

Tecnologias sugeridas: Node.js/TypeScript ou Dart (Shelf/Aqueduct).
